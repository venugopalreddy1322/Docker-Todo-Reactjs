## Features in the application
- Dark mode
- Drag n drop tasks to reorder them
- Tasks saved locally
 Customisable task color
- Edit task
- Notification Box
- Tablet & mobile friendly

## Built with

- React Js, Css, Sass
- React Hooks : useState, useEffect, useRef, useContext
- react-beautiful-dnd, react-icons, uuidv4...
